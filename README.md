# MathExerciseApp
A simple JavaFX-based Application where you can show your mental arithmetic skills.
This App was build using the View-ViewModel-Model approach.

### How to use the app
- At the beginning, you can select the operators (+, - , * /) the number of tasks and the size of the numbers.
- Then you are shown math problems based on the settings you choose at the beginning.
- You have to type in the results as fast as possible and then press "Enter".
- After that, the next Task appears and the background-color tells you whether your result was correct or not.
- At the End, you see how long you have taken to solve all tasks and how many answers you solved correctly.
